# 「時間設計論」産学連携候補リスト

調査日: 2026-03-24

---

## 優先度A: 最初にアプローチすべき候補

### 1. 一川誠（千葉大学 大学院人文科学研究院 教授）-- 最有力
- **専門**: 実験心理学、知覚認知、時間知覚
- **代表的研究**: 『大人の時間はなぜ短いのか』『時計の時間、心の時間』
- **肩書**: 日本時間学会 会長 / 山口大学時間学研究所 客員教授
- **なぜ最有力か**: 自ら「時間デザイン」という概念を提唱しており、講座タイトル「時間設計論」との親和性が極めて高い
- **関連する講義回**: 第1回（時間の正体）、第2回（集中と時間知覚）
- **連携形態**: ゲスト講師、共同企画、日本時間学会での発表
- **コンタクト**: [千葉大学教員ページ](https://www.l.chiba-u.jp/applicants/teachers/makoto_ichikawa.html) / [researchmap](https://researchmap.jp/ichikawa)

### 2. 伊達洋駆（ビジネスリサーチラボ代表 / 東京大学特任研究員）
- **専門**: 組織心理学、人と組織の行動科学
- **代表的研究**: 『人と組織の行動科学』、注意残余のビジネス解説を多数発信
- **なぜ有力か**: 学術知見のビジネス翻訳に長けている。民間シンクタンクで小回りが利き、連携ハードルが低い
- **関連する講義回**: 第2回（注意残余の実務的インパクト）
- **連携形態**: 講義内容の監修・共同設計、ゲスト講師
- **コンタクト**: [ビジネスリサーチラボ](https://www.business-research-lab.com/fellow/date/)

### 3. 山口大学 時間学研究所（RITS）-- 機関レベル連携
- **所長**: 藤澤健太 教授（宇宙物理学）
- **概要**: 2000年設立。7部門（宇宙・地球科学、数理科学、生命科学、工学、心理学、社会科学、人文科学）に52名の研究員。文理横断の「時間学」研究機関として世界的にも希少
- **関連する講義回**: 全回（「時間学」フレームワーク自体が講座コンセプトと合致）
- **連携形態**: 包括連携協定、TimeCrowdデータの研究提供、日本時間学会での共同発表、「ビジネスにおける時間設計」分野の開拓
- **コンタクト**: [時間学研究所](http://www.rits.yamaguchi-u.ac.jp/)

## 優先度B: 正式な産学連携として

### 4. 大湾秀雄（早稲田大学 政治経済学術院 教授）
- **専門**: 人事経済学、組織経済学、ピープルアナリティクス
- **代表的研究**: 『日本の人事を科学する』、ピープルアナリティクスアカデミー共催
- **関連する講義回**: 第4回（時間データの見える化）
- **連携形態**: TimeCrowdの時間データを使った人事データ分析の共同研究、ピープルアナリティクスアカデミーへの講座提供
- **コンタクト**: [研究室HP](https://hideo-owan.net/) / [早稲田大学](https://w-rdb.waseda.jp/html/100001279_ja.html)

### 5. 島津明人（慶應義塾大学 総合政策学部 教授）
- **専門**: 産業保健心理学、ワーク・エンゲイジメント
- **代表的研究**: 日本におけるワーク・エンゲイジメント研究の第一人者。UWES日本版を開発
- **研究室テーマ**: 「3つのHP：健康増進、生産性向上、幸福」
- **関連する講義回**: 第2回（集中とエンゲイジメント）、第4回（計測と改善）
- **連携形態**: TimeCrowdデータ×ワーク・エンゲイジメント尺度の共同研究
- **コンタクト**: [島津研究室](https://hp3.jp/)

### 6. 山本勲（慶應義塾大学 商学部 教授）
- **専門**: 労働経済学（労働時間、働き方改革、健康経営、生産性）
- **代表的研究**: 『労働時間の経済分析』（日経・経済図書文化賞受賞）、RIETI「働き方改革と健康経営」PJリーダー
- **関連する講義回**: 第1回（労働時間と生産性）、第4回（時間データ）
- **連携形態**: ゲスト講師、TimeCrowdデータを用いた労働時間効果の実証研究
- **コンタクト**: [慶應義塾大学](https://k-ris.keio.ac.jp/html/100001120_ja.html) / [researchmap](https://researchmap.jp/read0138102)

## 推奨アクションプラン

**Phase 1: すぐ着手**
1. 一川誠教授にコンタクト -- 時間学会会長×「時間デザイン」提唱者。講座コンセプトとの一致度が最高
2. 伊達洋駆氏にコンタクト -- 民間なので連携が早い。注意残余のビジネス文脈での解説実績あり

**Phase 2: 正式連携**
3. 山口大学時間学研究所 -- 機関レベルの包括連携。「ビジネスにおける時間設計」という新分野
4. 大湾秀雄教授 -- ピープルアナリティクスの文脈でTimeCrowdデータ活用を提案

**Phase 3: 権威付け**
5. 山本勲教授 or 鶴光太郎教授（慶應） -- 労働時間研究の第一人者
6. 森川正之特任教授（一橋/RIETI） -- 生産性研究の大家
