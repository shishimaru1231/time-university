よ# Time University 事業構想 -- 深掘りメモ

## ビジョン（和田メモより）

1. タイムトラッキングの会社 → 時間管理のスペシャリスト集団に変革する
2. 日本の時間管理を「した方が良いもの」→「しなくては仕事が回らないもの」へパラダイムシフト
3. 時間管理のスペシャリストを定義し、資格を作る
4. 資格を取るための講座を作る = Time University
5. 上記をPRし、日本で時間管理に最も専門性を持つ民間企業として確固たる地位を築く

## 突き詰めるべき問い

### 1. 資格の設計

**Q: 誰が取る資格か？**
- A案: 個人のビジネスパーソン（セルフマネジメント能力の証明）
- B案: 企業の人事・管理職（組織の時間設計スキルの証明）
- C案: コンサルタント・研修講師（時間管理を教える側の資格）
- D案: 上記すべてのレベル分け

**Q: 資格の名前は？**
- 案1: 時間管理スペシャリスト（Time Management Specialist / TMS）
- 案2: 時間設計士（Time Design Professional / TDP）
- 案3: タイムエンジニア（Time Engineer / TE）
- 案4: 時間管理マイスター

**Q: 資格の権威付けはどう担保する？**
- 産学連携（大学教授の監修・試験委員会への参画）
- 学会との連携（日本時間学会、ドラッカー学会）
- 受験者の実績・事例の蓄積
- 先行事例: ITパスポート、PMP、簿記 → 民間資格から始まり社会的認知を得たもの

**Q: 資格のレベル分けは？**
- Level 1: 基礎（個人の時間管理 -- School 1-2相当）
- Level 2: 応用（組織の時間設計 -- School 3-4相当）
- Level 3: 専門（AI時代の時間再配分+教える側 -- School 5+教授法）

### 2. パラダイムシフトの起こし方

**Q: 「時間管理をしなければ仕事が回らない」状態をどう作るか？**

現状の「した方が良い」が変わらない理由:
- 時間管理しなくても目に見える損失がない（ように見える）
- 時間管理の効果が数値化されていない
- 時間管理を教える場所がない → スキルとして認識されていない

パラダイムシフトのレバー:
- **資格制度**: 「時間管理はスキルである」という認識を作る。スキルには資格がある
- **エビデンス**: 学術研究に基づく「時間管理しないことの損失」の数値化
- **事例**: 資格取得企業の生産性改善データの蓄積と公開
- **制度的後押し**: 働き方改革関連法、健康経営優良法人との連携

**類似のパラダイムシフト事例**:
- 情報セキュリティ: 「やった方がいい」→「ISMSがないと取引できない」
- 個人情報保護: 「気をつけよう」→「Pマークがないと信用されない」
- メンタルヘルス: 「元気にやれ」→「ストレスチェック義務化」
- 会計: 「どんぶり勘定でも回る」→「簿記は経営の基礎」

### 3. タイムクラウド社の変革

**Q: 「タイムトラッキングの会社」から「時間管理のスペシャリスト集団」への変革は何を意味するか？**

変わること:
- 提供価値: ツール → ツール+知識+資格+コンサルティング
- 収益構造: SaaS月額 → SaaS+資格受験料+講座受講料+企業研修+コンサルティング
- 人材要件: エンジニア中心 → エンジニア+教育コンテンツ制作+認定講師+研究者
- ブランド: 「便利なツール」→「時間管理の権威」
- 競合: 他の工数管理ツール → 競合不在（時間管理の資格を持つ会社は他にない）

### 4. Time Universityの位置づけ変更

**Before（マーケティング施策としてのTU）**:
- 全コンテンツ無料
- 目的: 会員を集めてTimeCrowdに誘導
- KPI: 会員数、トライアル転換率

**After（資格教育機関としてのTU）**:
- 基礎コンテンツは無料（認知拡大・市場教育）
- 資格取得講座は有料（収益源）
- 目的: 時間管理スペシャリストを育成し、その基盤としてTimeCrowdが使われる
- KPI: 資格取得者数、企業導入数、メディア露出

### 5. 収益モデルの再設計

| 収益源 | 単価感 | 規模感 |
|--------|--------|--------|
| TimeCrowd SaaS | 月額/人 | 既存事業 |
| 資格受験料 | 1-3万円/回 | 年間受験者数 x 単価 |
| 資格取得講座 | 5-15万円/コース | 受講者数 x 単価 |
| 企業研修 | 30-100万円/回 | 企業数 x 単価 |
| 認定講師養成 | 30-50万円/人 | 講師数 x 単価 |
| 教材・書籍 | 2,000-5,000円 | 販売数 x 単価 |
| 更新料（資格の有効期限） | 5,000-1万円/年 | 資格保有者数 x 単価 |

## 実行の順序（仮説）

### Phase 0: 土台作り（今〜Q3 2026）
- Time Universityの無料コンテンツで認知拡大
- 産学連携の開始（一川教授、伊達氏）
- 「時間管理スペシャリスト」の定義・スキル体系の設計
- 資格試験の問題設計（学術パートナーと共同）

### Phase 1: 資格のローンチ（Q4 2026〜Q1 2027）
- Level 1 資格試験の開始
- 資格取得講座の提供開始
- 第1期合格者の輩出 → PR
- 企業向け研修プログラムの開始

### Phase 2: 権威の確立（2027〜）
- Level 2, 3 の追加
- 資格取得者の実績データ蓄積・公開
- メディア露出の強化
- 日本時間学会での発表
- 「時間管理の会社といえばタイムクラウド」の認知確立

## 次に決めるべきこと

1. **資格の対象者**: 誰が取る資格なのかを決める
2. **資格の名前**: ブランドとして成立する名前を決める
3. **スキル体系**: 「時間管理スペシャリスト」が持つべきスキルセットを定義する
4. **試験設計**: 何を・どう測るかを決める
5. **価格設定**: 受験料・講座料金を決める
6. **ロードマップ**: Phase 0で何をいつまでにやるかを決める
