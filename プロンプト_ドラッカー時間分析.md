# ドラッカー式 時間分析プロンプト

TimeCrowdの時間データ（または任意の時間記録）を、ドラッカー『経営者の条件』第2章「汝の時間を知れ」の3ステップで分析するプロンプト。

---

## 使い方

1. TimeCrowdから期間を指定してCSVエクスポート（推奨: 2-4週間分）
2. 以下のプロンプトにデータを添付して実行
3. Claude / ChatGPT / NotebookLM いずれでも使用可

---

## プロンプト

```
あなたはドラッカー『経営者の条件』第2章「汝の時間を知れ」に基づく
時間分析の専門家です。

以下は私の[期間]の業務時間記録データです。
ドラッカーの3ステップに従って分析してください。

---

## Step 1: 記録の分析（Know Thy Time）

ドラッカーは「記憶ではなく記録で」と書きました。
データに基づいて以下を明らかにしてください。

- 時間の総量と日別の分布
- カテゴリ別の時間配分（上位5カテゴリとその割合）
- 「思考時間」と「作業時間」の比率
  - 思考時間: 企画、戦略立案、意思決定、創造的作業
  - 作業時間: 定型業務、メール対応、会議、事務処理
- 最も時間を使っているのに成果が不明確な活動
- 時間の使い方で驚くべきパターン（本人が気づいていないもの）

## Step 2: 整理の提案（Manage Thy Time）

ドラッカーは3つの問いで時間を整理せよと書きました。
データの中から以下に該当するものを特定してください。

### 問い1:「まったくしなかったら何が起こるか？」
→ やめても何も起こらない活動を特定してください。

### 問い2:「他の人でもできることはないか？」
→ 本人でなくてもできる（委任可能な）活動を特定してください。

### 問い3:「他人の時間を無駄にしていないか？」
→ 会議や依頼で他者の時間を浪費している可能性がある活動を特定してください。

それぞれについて、具体的な活動名と推定削減可能時間を示してください。

## Step 3: まとめの設計（Consolidate Thy Time）

ドラッカーは「自由になった時間を大きなまとまりに集約せよ」と書きました。
Step 2で削減可能と判定した時間を回収した場合:

- 週あたり何時間の「自由裁量時間」が生まれるか
- その時間を使って取り組むべき「最も重要だが着手できていないこと」の提案
- 具体的な週間スケジュールの再設計案
  - 「思考のためのブロック」をどこに配置するか
  - 中断を最小化するための構造
  - Cal Newportのタイムブロッキングの考え方も参考に

---

## 出力形式

### サマリー（1段落）
この人の時間の使い方の最大の問題点を一言で。

### 詳細分析（Step 1-3）
上記の各ステップに従って。

### アクションプラン（3つだけ）
明日から始められる具体的な行動を3つ。
優先度の高い順に。

### ドラッカーからの引用
分析結果に最も合致する『経営者の条件』からの引用を1つ選んで締める。
```

---

## TimeCrowdデータの前処理（必要に応じて）

TimeCrowdのCSVエクスポートには以下のカラムがあるため、
プロンプトに添付する前にカテゴリの説明を補足すると精度が上がる。

```
追加コンテキスト（プロンプトの冒頭に添付）:

私の役職: [CRO / マネージャー / エンジニア 等]
チーム規模: [人数]
主な業務: [箇条書きで3-5個]
現在の最大の課題: [1つ]
「本当はやりたいのに時間がないこと」: [1つ]
```

---

## 応用: チーム版

```
以下は私のチーム（[人数]名）の[期間]の業務時間記録データです。
ドラッカーの視点に加え、Shangguan (2025) の知見
「チーム生産性向上の60%はチーム内の時間再配分による」を踏まえて分析してください。

- チーム全体の時間配分の偏り（特定メンバーへの集中度）
- チームとして「やめるべき活動」
- チームとして「まとめるべき時間」
- 時間再配分の提案（誰の何を誰に移すか）
```

---

## 応用: 定期レビュー版（月1回）

```
以下は私の先月の業務時間記録データです。
前月のデータ（添付）と比較して、以下を分析してください。

- 「思考時間」の比率は増えたか減ったか
- Step 2で特定した「やめるべき活動」は実際に減ったか
- 新たに発生した時間の浪費パターンはないか
- 「自由裁量時間」の実質量の変化
- 来月に向けた調整提案（1つだけ）

ドラッカーは「4-6週間ごとに記録を取り直せ」と書きました。
この定期レビューがその実践です。
```
